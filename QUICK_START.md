## 🚀 Quick Start - FitBody90 App

### Step 1: Install Dependencies
```bash
cd /Users/subhashraj/MyDocs/ReactNative/FitBody90/FitBody90
npm install
```

### Step 2: Start the Dev Server
```bash
npm start
```

You'll see a QR code in your terminal.

### Step 3: Open in Expo Go
- **iPhone**: Open Camera app, point at QR code, tap notification
- **Android**: Open Expo Go app, tap "Scan QR code", point at QR code

### Step 4: Start Using the App
1. Open **Dashboard** tab to see your day counter and daily checklist
2. Go to **Workout** tab to log your exercises
3. Use **Nutrition** tab to track calories, protein, and water
4. Check **Progress** tab to log your weight
5. View **Plan** tab for your 90-day roadmap

## 🎯 Default Targets
- **Goal Weight**: 72 kg (from 82 kg)
- **Daily Calories**: 1900-2100 kcal
- **Daily Protein**: 150g
- **Daily Water**: 3L
- **Daily Sleep**: 7 hours
- **Daily Steps**: 8,000

## ⚙️ Terminal Commands While App is Running

While the Expo server is running, you can:
- Press **`r`** to reload the app
- Press **`i`** to open iOS simulator
- Press **`a`** to open Android emulator
- Press **`w`** to open web version
- Press **`Ctrl+C`** to stop the server

## ❓ Troubleshooting

**App won't load?**
- Make sure you're on same WiFi as your computer
- Try pressing `r` in terminal to reload
- Kill server with `Ctrl+C` and run `npm start` again

**Data not saving?**
- Close and reopen Expo Go
- Check that you have storage space on your phone
